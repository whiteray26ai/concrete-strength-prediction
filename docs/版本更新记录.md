## [v1.0.0] - 2026-07-05 (初始版本)

### 新增

#### 核心功能模块
- **数据读取模块** (`src/data_reader.py`)
  - `DataReader` 类：支持CSV格式数据集读取
  - 自动分离特征列和目标列
  - 异常处理：文件不存在、数据格式错误

- **数据预处理模块** (`src/data_preprocessor.py`)
  - `DataPreprocessor` 类
  - Z-Score标准化：`standardize()` 方法
  - 训练集/测试集划分：`split()` 方法
  - 逆标准化：`inverse_standardize()` 方法

- **多元线性回归算法** (`src/linear_regression.py`)
  - `LinearRegression` 类
  - 基于梯度下降算法自行实现
  - 支持L2正则化防止过拟合
  - 可配置学习率、迭代次数、正则化系数

- **决策树回归算法** (`src/decision_tree.py`)
  - `DecisionTreeRegressor` 类
  - 基于CART算法自行实现
  - 使用MSE作为分裂准则
  - 可配置最大深度、最小样本数、最小MSE下降值

- **模型训练器** (`src/model_trainer.py`)
  - `ModelTrainer` 类
  - 支持多种模型类型（线性回归、决策树回归）
  - 统一的训练和预测接口

- **评估器** (`src/evaluator.py`)
  - `Evaluator` 类
  - 计算均方误差（MSE）
  - 计算均方根误差（RMSE）
  - 计算平均绝对误差（MAE）
  - 计算决定系数（R²）

- **结果导出器** (`src/result_exporter.py`)
  - `ResultExporter` 类
  - 导出评估报告（TXT格式）
  - 导出预测结果（CSV格式）
  - 自动生成带时间戳的文件名

- **GUI主界面** (`src/main_app.py`)
  - `MainApplication` 类（继承自 `tk.Tk`）
  - 基于tkinter的图形用户界面
  - 文件选择、参数配置、训练、结果展示、导出功能
  - matplotlib图表嵌入展示

#### 数据资源
- **示例数据集** (`data/concrete_data.csv`)
  - 251个样本，8个特征列，1个目标列
  - 包含水泥、矿渣、粉煤灰、水、减水剂、粗骨料、细骨料、龄期、抗压强度

## 版本说明

### 版本号规则
采用语义化版本号：`主版本号.次版本号.修订号`

- **主版本号**：不兼容的API修改
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修正

### 版本分类
- **v1.0.0**：初始版本，包含系统所有核心功能
- **v1.1.x**：文档和配置文件调整
- **v1.2.x**：功能优化和问题修复
