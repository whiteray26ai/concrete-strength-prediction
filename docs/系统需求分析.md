# 混凝土抗压强度预测系统 - 需求分析文档

## 1. 背景与目标

### 1.1 项目背景
在建筑工程设计与施工中，混凝土抗压强度是衡量结构安全的关键指标。传统方法依赖于实验室试块养护和破坏性试验，周期长、成本高。利用人工智能技术，基于混凝土配合比和龄期等特征，预测其抗压强度，可辅助工程人员快速评估混凝土质量，优化配比设计。

### 1.2 系统目标
开发一个完整的混凝土抗压强度预测系统，采用面向对象思想进行架构设计，集成数据读取、预处理、模型训练与评估、结果可视化与导出等模块。

## 4. 数据需求

### 4.1 数据集格式
CSV格式，包含以下列：
- **cement**: 水泥含量（kg/m³）
- **blast_furnace_slag**: 高炉矿渣含量（kg/m³）
- **fly_ash**: 粉煤灰含量（kg/m³）
- **water**: 水含量（kg/m³）
- **superplasticizer**: 减水剂含量（kg/m³）
- **coarse_aggregate**: 粗骨料含量（kg/m³）
- **fine_aggregate**: 细骨料含量（kg/m³）
- **age**: 龄期（天）
- **compressive_strength**: 抗压强度（MPa）- 目标变量

### 4.2 数据规模
示例数据集包含约250个样本，系统应支持扩展到更大规模数据集。

## 5. 约束条件

1. 系统使用Python语言开发，版本要求3.7及以上
2. 使用标准库和matplotlib进行可视化，不依赖其他第三方机器学习库
3. 核心算法需自行实现，体现对算法原理的理解
4. 界面使用tkinter，不使用复杂动画效果
