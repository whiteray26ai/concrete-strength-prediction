## 1. 系统架构设计

### 1.1 分层架构
系统采用三层架构设计，分为表示层、业务逻辑层和数据访问层。

```
┌─────────────────────────────────────────────────────┐
│                   表示层 (UI)                       │
│            MainApplication (tkinter GUI)            │
├─────────────────────────────────────────────────────┤
│                 业务逻辑层 (Logic)                   │
│  DataPreprocessor  ModelTrainer  Evaluator          │
│     数据预处理       模型训练       模型评估          │
├─────────────────────────────────────────────────────┤
│                 数据访问层 (Data)                    │
│      DataReader          ResultExporter             │
│      数据读取              结果导出                  │
├─────────────────────────────────────────────────────┤
│                   核心算法层                         │
│    LinearRegression      DecisionTreeRegressor      │
│      多元线性回归           决策树回归               │
└─────────────────────────────────────────────────────┘
```

### 1.2 各层职责

| 层级 | 职责描述 |
| :--- | :--- |
| 表示层 | 负责用户交互，调用业务逻辑，展示结果 |
| 业务逻辑层 | 数据预处理、模型训练、评估等核心功能 |
| 数据访问层 | 从文件读取数据和将结果写入文件 |
| 核心算法层 | 实现具体的机器学习算法 |

## 3. 数据结构设计

### 3.1 数据集存储
- **特征数组 (X)**: 二维列表，每行一个样本，每列一个特征
- **目标数组 (y)**: 一维列表，每个元素对应一个样本的目标值
- **特征名称 (feature_names)**: 列表，存储各特征的名称

### 3.2 模型参数
- **线性回归**: 字典 `{'theta': [...], 'learning_rate': 0.01, 'max_iter': 1000, 'alpha': 0.0}`
- **决策树**: 字典 `{'max_depth': 5, 'min_samples_split': 2, 'min_mse_decrease': 0.0}`

### 3.3 评估指标
- 字典 `{'MSE': 125.3, 'RMSE': 11.2, 'MAE': 8.5, 'R2': 0.82}`

## 4. 核心算法设计

### 4.1 多元线性回归（梯度下降）

**损失函数**:
```
J(θ) = (1/2m) * Σ(h(x) - y)^2 + (α/2) * Σ(θ_j^2)
```

**梯度下降更新**:
```
θ_j = θ_j - α * (1/m) * Σ((h(x) - y) * x_j) - α * θ_j  (j > 0)
```

**参数**:
- learning_rate: 学习率，默认0.01
- max_iter: 最大迭代次数，默认1000
- alpha: L2正则化系数，默认0.0

### 4.2 决策树回归（CART）

**分裂准则**: 选择使MSE最小的特征和分裂点

**停止条件**:
1. 达到最大深度
2. 节点样本数小于阈值
3. MSE降低小于阈值

**预测方式**: 遍历树到叶子节点，返回叶子节点目标均值

### 4.3 数据标准化（Z-Score）

```
x' = (x - μ) / σ
```

其中 μ 为特征均值，σ 为特征标准差

### 5.2 交互流程

1. 用户点击"打开数据文件"选择CSV文件
2. 系统加载数据并显示样本信息
3. 用户选择模型类型和参数
4. 用户调整训练集/测试集比例
5. 用户点击"开始训练"按钮
6. 系统进行数据预处理、模型训练和评估
7. 系统展示预测结果图表和评估指标
8. 用户可点击"导出结果"保存报告和预测值

```python
class DataReader:
```

### 6.2 DataPreprocessor

```python
class DataPreprocessor:
    def standardize(self, X: List[List[float]]) -> List[List[float]]
    def split(self, X: List[List[float]], y: List[float], test_ratio: float, random_seed: int) -> Tuple[List[List[float]], List[List[float]], List[float], List[float]]
```

### 6.3 ModelTrainer

```python
class ModelTrainer:
    def train(self, X_train: List[List[float]], y_train: List[float]) -> None

```

### 6.4 Evaluator

```python
class Evaluator:
    def evaluate(self, y_true: List[float], y_pred: List[float]) -> Dict[str, float]
```

```python
class ResultExporter:
```
