# 混凝土抗压强度预测系统 - 详细设计文档

## 1. 数据读取模块 (data_reader.py)

### 1.1 类定义

```python
class DataReader:
    def __init__(self, file_path=None):
        self.file_path = file_path
        self.feature_names = []
        self.target_name = ''
```

### 1.2 方法实现

**load_data(file_path)**
- 功能：从CSV文件读取数据
- 参数：file_path - CSV文件路径
- 返回：(X, y, feature_names) - 特征数组、目标数组、特征名称
- 异常处理：文件不存在、数据格式错误

**实现逻辑**：
1. 验证文件路径
2. 打开CSV文件，读取表头
3. 解析每行数据，转换为浮点数
4. 分离特征列和目标列
5. 返回处理后的数据

### 1.3 数据格式要求
- CSV文件必须包含表头
- 最后一列为目标变量（抗压强度）
- 前8列为特征变量
- 数据必须为数值类型

## 2. 数据预处理模块 (data_preprocessor.py)

### 2.1 类定义

```python
class DataPreprocessor:
    def __init__(self):
        self.scaler_params = {}
```

### 2.2 方法实现

**standardize(X)**
- 功能：Z-Score标准化
- 参数：X - 特征数组（二维列表）
- 返回：标准化后的特征数组

**实现逻辑**：
1. 计算每列特征的均值 μ 和标准差 σ
2. 对每个特征值进行转换：x' = (x - μ) / σ
3. 保存标准化参数用于逆变换

**split(X, y, test_ratio, random_seed)**
- 功能：划分训练集和测试集
- 参数：
  - X: 特征数组
  - y: 目标数组
  - test_ratio: 测试集比例
  - random_seed: 随机种子
- 返回：(X_train, X_test, y_train, y_test)

**实现逻辑**：
1. 验证特征和目标长度一致
2. 生成随机索引序列
3. 根据比例划分训练集和测试集

**inverse_standardize(X_std)**
- 功能：逆标准化
- 参数：X_std - 标准化后的特征数组
- 返回：原始特征数组

## 3. 线性回归模块 (linear_regression.py)

### 3.1 类定义

```python
class LinearRegression:
    def __init__(self, learning_rate=0.01, max_iter=1000, alpha=0.0):
        self.learning_rate = learning_rate
        self.max_iter = max_iter
        self.alpha = alpha
        self.theta = None
```

### 3.2 方法实现

**_add_bias(X)**
- 功能：为特征矩阵添加偏置列
- 参数：X - 特征数组
- 返回：添加偏置后的特征数组

**_predict(X, theta)**
- 功能：计算预测值
- 参数：
  - X: 添加偏置后的特征数组
  - theta: 参数向量
- 返回：预测值数组

**_gradient_descent(X_bias, y)**
- 功能：梯度下降算法
- 参数：
  - X_bias: 添加偏置后的特征数组
  - y: 目标数组

**实现逻辑**：
1. 初始化参数向量 θ 为全零
2. 迭代max_iter次：
   - 计算当前预测值
   - 计算梯度
   - 更新参数向量（含正则化）

**train(X, y)**
- 功能：训练模型
- 参数：
  - X: 特征数组
  - y: 目标数组

**predict(X)**
- 功能：预测目标值
- 参数：X - 特征数组
- 返回：预测值数组


### 4.2 方法实现

**_compute_mse(y)**
- 功能：计算MSE
- 参数：y - 目标值数组
- 返回：MSE值

**_find_best_split(X, y)**
- 功能：寻找最佳分裂特征和分裂点
- 参数：
  - X: 特征数组
  - y: 目标数组
- 返回：(best_feature_idx, best_split_val, best_mse)

**实现逻辑**：
1. 遍历所有特征
2. 对每个特征的所有可能分裂点计算MSE
3. 选择MSE最小的分裂点

**_build_tree(X, y, depth)**
- 功能：递归构建决策树
- 参数：
  - X: 特征数组
  - y: 目标数组
  - depth: 当前深度
- 返回：树节点（字典）

**实现逻辑**：
1. 检查停止条件（深度、样本数、MSE下降）
2. 寻找最佳分裂点
3. 递归构建左右子树
4. 返回树结构

**train(X, y)**
- 功能：训练决策树模型
- 参数：
  - X: 特征数组
  - y: 目标数组

**_predict_single(x, tree)**
- 功能：对单个样本预测
- 参数：
  - x: 单个样本特征

- 返回：预测值

**predict(X)**
- 功能：预测目标值
- 参数：X - 特征数组
- 返回：预测值数组

## 6. 评估器模块 (evaluator.py)

### 6.1 类定义

```python
class Evaluator:
    def __init__(self):
        self.metrics = {}
```

### 6.2 方法实现

**_compute_mse(y_true, y_pred)**
- 功能：计算均方误差
- 公式：MSE = (1/n) * Σ(y_true - y_pred)^2

**_compute_rmse(y_true, y_pred)**
- 功能：计算均方根误差
- 公式：RMSE = sqrt(MSE)

**_compute_mae(y_true, y_pred)**
- 功能：计算平均绝对误差
- 公式：MAE = (1/n) * Σ|y_true - y_pred|

**_compute_r2(y_true, y_pred)**
- 功能：计算决定系数
- 公式：R² = 1 - SS_res / SS_tot

**evaluate(y_true, y_pred)**
- 功能：计算所有评估指标
- 参数：
  - y_true: 真实值数组
  - y_pred: 预测值数组
- 返回：指标字典

**get_report()**
- 功能：获取评估报告文本
- 返回：格式化的报告字符串

## 9. 算法流程图

### 9.1 梯度下降算法流程图

```
开始
  │
  ▼
初始化 θ = 0
  │
  ▼
for i = 1 to max_iter:
  │
  ├─▶ 计算预测值 h(x) = Xθ
  │
  ├─▶ 计算梯度 ∇J = (1/m) * X^T (Xθ - y)
  │
  ├─▶ 添加正则化项（j > 0）
  │
  └─▶ 更新 θ = θ - α * ∇J
  │
  ▼
结束，返回 θ
```

### 9.2 决策树构建流程图

```
开始
  │
  ▼
检查停止条件？
  │
  ├─是─▶ 返回叶子节点（目标均值）
  │
  └─否─▶ 寻找最佳分裂点
  │
  ├─无分裂点─▶ 返回叶子节点
  │
  └─有分裂点─▶ 划分左右子树
  │
  ├─▶ 递归构建左子树
  │
  ├─▶ 递归构建右子树
  │
  └─▶ 返回分裂节点
  │
  ▼
结束
```
