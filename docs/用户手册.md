# 混凝土抗压强度预测系统 - 用户手册

## 1. 系统简介

### 1.1 系统概述
本系统是一个基于机器学习的混凝土抗压强度预测系统，采用面向对象思想设计，集成了数据读取、预处理、模型训练与评估、结果可视化与导出等功能。系统支持多元线性回归和决策树回归两种算法，可帮助用户快速预测混凝土抗压强度。

### 1.2 主要功能
- 数据加载：支持CSV格式的数据集
- 数据预处理：标准化特征、划分训练集与测试集
- 模型训练：多元线性回归、决策树回归
- 结果评估：计算MSE、RMSE、MAE、R²等指标
- 结果展示：真实值vs预测值散点图、残差分布图
- 结果导出：导出评估报告和预测结果

### 1.3 系统要求

| 项目 | 要求 |
| :--- | :--- |
| 操作系统 | Windows 7/10/11 |
| Python版本 | 3.7及以上 |
| 内存 | 至少4GB |
| 屏幕分辨率 | 至少1024x768 |

## 2. 安装与启动

### 2.1 安装步骤

1. **下载源代码**
   - 将项目文件解压到本地目录，例如：`C:\concrete_prediction`

2. **安装依赖**
   - 打开命令行，进入项目目录
   - 执行以下命令安装依赖：
   ```
   pip install matplotlib
   ```

### 2.2 启动系统

1. **方法一：双击运行**
   - 双击 `main.py` 文件

2. **方法二：命令行运行**
   - 打开命令行，进入项目目录
   - 执行以下命令：
   ```
   python main.py
   ```

3. **启动成功**
   - 系统界面会显示，标题为"混凝土抗压强度预测系统"

## 3. 操作指南

### 3.1 加载数据

1. 点击界面顶部的 **"打开数据文件"** 按钮
2. 在弹出的文件选择对话框中，选择CSV格式的数据文件
3. 系统会自动加载数据，并在界面右上角显示样本数和特征数
4. 如果加载成功，"开始训练"按钮会变为可用状态

**数据格式要求**：
- CSV文件必须包含表头
- 最后一列为目标变量（抗压强度）
- 前8列为特征变量（水泥、矿渣、粉煤灰、水、减水剂、粗骨料、细骨料、龄期）
- 数据必须为数值类型

**示例数据格式**：
```csv
cement,blast_furnace_slag,fly_ash,water,superplasticizer,coarse_aggregate,fine_aggregate,age,compressive_strength
540,0,0,162,2.5,1040,676,28,79.99
540,0,0,162,2.5,1055,676,28,61.89
```

### 3.2 选择模型

系统提供两种机器学习算法：

1. **多元线性回归**
   - 基于梯度下降的线性回归算法
   - 支持L2正则化

2. **决策树回归**
   - 基于CART算法的决策树回归
   - 通过最小化MSE进行分裂

选择方法：
- 在左侧"参数设置"面板中，点击对应的单选按钮
- 参数面板会根据选择的模型类型自动更新

### 3.3 配置参数

#### 3.3.1 多元线性回归参数

| 参数 | 说明 | 默认值 | 建议范围 |
| :--- | :--- | :--- | :--- |
| 学习率 | 梯度下降的步长 | 0.01 | 0.001-0.1 |
| 迭代次数 | 梯度下降的迭代次数 | 1000 | 500-5000 |
| 正则化系数 | L2正则化参数，0表示不使用 | 0.0 | 0-1 |

**参数调整建议**：
- 如果模型不收敛，尝试减小学习率
- 如果收敛速度慢，尝试增加迭代次数
- 如果过拟合，尝试增加正则化系数

#### 3.3.2 决策树回归参数

| 参数 | 说明 | 默认值 | 建议范围 |
| :--- | :--- | :--- | :--- |
| 最大深度 | 决策树的最大深度 | 5 | 3-15 |
| 最小样本数 | 节点分裂所需的最小样本数 | 2 | 2-10 |

**参数调整建议**：
- 如果过拟合，尝试减小最大深度
- 如果欠拟合，尝试增加最大深度

### 3.4 设置数据划分

1. 在左侧面板的"数据划分"区域，调整训练集比例滑动条
2. 滑动条范围为50%-95%，默认为80%
3. 测试集比例 = 1 - 训练集比例

**建议**：训练集比例设置为70%-80%较为合适

### 3.5 选择数据预处理

在"数据预处理"区域，勾选"标准化特征"选项：
- 标准化会将特征转换为均值为0、标准差为1的分布
- 建议勾选，特别是对于线性回归算法
- 标准化有助于加速梯度下降收敛

### 3.6 开始训练

1. 确认所有参数设置完成
2. 点击底部的 **"开始训练"** 按钮
3. 系统会依次执行：
   - 数据标准化（如果选择）
   - 划分训练集和测试集
   - 训练模型
   - 预测测试集
   - 计算评估指标
4. 训练完成后，右侧面板会显示：
   - 真实值vs预测值散点图
   - 残差分布图
   - 评估指标（MSE、RMSE、MAE、R²）

### 3.7 查看结果

#### 3.7.1 图表解读

**真实值vs预测值散点图**：
- 横坐标：真实的抗压强度值
- 纵坐标：模型预测的抗压强度值
- 红色虚线：y=x，理想情况下所有点应落在这条线上
- 点越接近红色虚线，说明预测越准确

**残差分布图**：
- 横坐标：残差（真实值 - 预测值）
- 纵坐标：频率
- 理想情况下，残差应呈正态分布，均值接近0

#### 3.7.2 评估指标解读

| 指标 | 说明 | 理想值 |
| :--- | :--- | :--- |
| MSE | 均方误差，衡量预测值与真实值的平均平方差 | 越小越好 |
| RMSE | 均方根误差，MSE的平方根 | 越小越好 |
| MAE | 平均绝对误差，衡量预测值与真实值的平均绝对差 | 越小越好 |
| R² | 决定系数，衡量模型解释方差的比例 | 越接近1越好 |

**指标判断标准**：
- R² > 0.7：模型拟合较好
- 0.5 < R² ≤ 0.7：模型拟合一般
- R² ≤ 0.5：模型拟合较差，需要调整参数或更换算法

### 3.8 导出结果

1. 训练完成后，点击底部的 **"导出结果"** 按钮
2. 系统会自动生成两个文件：
   - 评估报告（TXT格式）：包含模型类型、参数和评估指标
   - 预测结果（CSV格式）：包含真实值、预测值和残差
3. 文件保存在项目目录的 `output` 文件夹中
4. 文件名包含时间戳，例如：`report_linear_regression_20240101_120000.txt`

### 3.9 清除结果

点击底部的 **"清除"** 按钮，系统会：
- 清空所有加载的数据
- 重置所有参数为默认值
- 清空图表和评估指标
- 界面恢复到初始状态

## 4. 使用示例

### 4.1 示例1：使用线性回归预测

1. 加载 `data/concrete_data.csv` 文件
2. 选择"多元线性回归"模型
3. 设置参数：学习率=0.1，迭代次数=1000，正则化=0.01
4. 设置训练集比例=80%
5. 勾选"标准化特征"
6. 点击"开始训练"
7. 查看结果：
   - MSE ≈ 111
   - RMSE ≈ 10.5
   - R² ≈ 0.17

### 4.2 示例2：使用决策树回归预测

1. 加载 `data/concrete_data.csv` 文件
2. 选择"决策树回归"模型
3. 设置参数：最大深度=5，最小样本数=2
4. 设置训练集比例=80%
5. 勾选"标准化特征"
6. 点击"开始训练"
7. 查看结果：
   - MSE ≈ 51
   - RMSE ≈ 7.2
   - R² ≈ 0.62

### 4.3 示例3：调整决策树深度

1. 按照示例2的步骤，将最大深度调整为10
2. 重新训练模型
3. 查看结果：
   - MSE ≈ 45
   - RMSE ≈ 6.7
   - R² ≈ 0.67
4. 结果显示，增加深度可以提高模型性能

## 5. 常见问题

### 5.1 数据加载失败

**问题描述**：点击"打开数据文件"后提示错误

**可能原因**：
- 文件不存在
- 文件格式不是CSV
- 文件内容格式错误

**解决方法**：
- 确认文件路径正确
- 确认文件是CSV格式（.csv扩展名）
- 确认文件包含表头和数值数据

### 5.2 模型训练失败

**问题描述**：点击"开始训练"后提示错误

**可能原因**：
- 数据未加载
- 参数设置不合理
- 数据格式错误

**解决方法**：
- 先加载数据再训练
- 检查参数是否在合理范围内
- 确认数据文件格式正确

### 5.3 预测结果不佳

**问题描述**：R²值较低，预测误差较大

**可能原因**：
- 模型选择不合适
- 参数设置不合理
- 数据量不足
- 特征质量差

**解决方法**：
- 尝试更换模型（线性回归→决策树回归）
- 调整模型参数
- 增加更多训练数据
- 检查特征是否与目标相关

### 5.4 界面显示异常

**问题描述**：界面显示不完整或图表无法显示

**可能原因**：
- 屏幕分辨率过低
- matplotlib安装异常

**解决方法**：
- 调整屏幕分辨率至1024x768以上
- 重新安装matplotlib：`pip install --upgrade matplotlib`
