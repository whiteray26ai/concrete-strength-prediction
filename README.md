# 混凝土抗压强度预测系统

基于机器学习的混凝土抗压强度预测系统，采用面向对象思想设计，集成数据读取、预处理、模型训练与评估、结果可视化与导出等模块。

## 功能特性

- **数据加载**：支持CSV格式的混凝土数据集读取
- **数据预处理**：Z-Score标准化、训练集/测试集划分
- **机器学习算法**：
  - 多元线性回归（基于梯度下降，支持L2正则化）
  - 决策树回归（基于CART算法，使用MSE分裂准则）
- **模型评估**：MSE、RMSE、MAE、R²等指标
- **可视化展示**：真实值vs预测值散点图、残差分布图
- **结果导出**：评估报告(TXT)、预测结果(CSV)
- **GUI界面**：基于tkinter的图形用户界面

## 环境要求

- Python 3.7+
- matplotlib（用于可视化）

## 安装与运行

1. 安装依赖：
```bash
pip install matplotlib
```

2. 运行系统：
```bash
python main.py
```

## 使用方法

1. 点击"打开数据文件"按钮，选择 `data/concrete_data.csv`
2. 选择模型类型（多元线性回归 / 决策树回归）
3. 配置模型参数
4. 调整训练集/测试集比例
5. 点击"开始训练"按钮
6. 查看预测结果和评估指标
7. 点击"导出结果"保存报告
